# Open Source Plan (30 Days)

## Goal by Day 30

- 10-minute local bootstrap
- real retry lifecycle demo (old run -> new retry run)
- contribution-ready repo (templates, labels, roadmap)
- weekly release rhythm

## Week 1 — Foundation

- repo hygiene + license + docs baseline
- docker compose quick start
- env templates and startup scripts
- first alpha tag

## Week 2 — Usability

- stable run/task/audit linkage rules
- end-to-end demo scenarios
- API docs for events/ops/retry
- v0.2 release

## Week 3 — Community Onboarding

- issue/PR templates
- contribution guide and good-first-issues
- architecture docs and diagrams
- v0.3 release

## Week 4 — Hardening + Growth

- fix top community pain points
- improve rollback and incident docs
- publish 90-day roadmap
- v0.4 release

## Product Boundary

### Open Source Core
- dashboard + APIs
- run/task/audit data model
- openclaw adapter endpoints
- self-host operations baseline

### Future Cloud / Enterprise
- managed hosting + SLA
- SSO and advanced RBAC
- org-level policy orchestration
- support and implementation package
