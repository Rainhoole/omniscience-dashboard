# Code of Conduct

## Our Standard

We build in public. Be respectful, direct, and constructive.

Expected behavior:
- discuss ideas, not personal attacks
- assume good intent, ask before escalating
- give actionable feedback
- welcome newcomers

Unacceptable behavior:
- harassment, hate speech, doxxing
- threats, intimidation, or repeated hostility
- sharing private credentials or sensitive user data

## Enforcement

Project maintainers may edit/remove comments, close threads, or ban contributors for serious violations.

For incidents, open a private report to maintainers.
